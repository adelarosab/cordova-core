## &lt;cordova-core&gt;

`<cordova-core>` implements a basic interface about cordova application status.

```html
<cordova-core
    ready
    paused
></cordova-core>
```
### Attributes
 
#### paused (read-only)
Return if cordova application is paused.

#### ready (read-only)
Return if cordova deviceready event has been fired.
